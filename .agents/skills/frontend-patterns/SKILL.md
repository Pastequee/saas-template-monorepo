# Frontend Patterns

Use this skill when implementing UI pages, hooks, queries, mutations, and forms in `apps/web`.

## What To Load

- Frontend architecture and implementation patterns: `references/frontend-patterns.md`

## Workflow

1. Follow file placement and organization rules from `references/frontend-patterns.md`.
2. Use Eden option factories for queries and mutations.
3. Keep query keys centralized.
4. Use `useAppForm` and shared form field/form components for forms.
