# Context

## Terms

### Listing

A user-owned marketplace record with title, description, price, and an optional image.

### Asset

A persisted record for an uploaded file. An Asset has an owner, a storage key, file metadata, and a lifecycle status.

### Asset lifecycle

The module that governs how an Asset is reserved for upload, activated after upload, attached to a Listing, retired when replaced or deleted, and cleaned up when stale.
